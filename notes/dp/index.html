What is Dynamic Programming?

DP is an optimization technique.

Dynamic Programming (DP) is used to solve an optimization problem by breaking it down into smaller subproblems and solution of problem will depend upon the solution of the subproblem.

Requirements of Dynamic Programming

  1. Overlapping Subproblem
  2. Optimal Substructure

Methods to solve Dynamic Programming

  1. Top-down (Memoization)
  2. Bottom-up (Tabulation)

Steps to Solve DP Problems

  1. Find the recurrence relation
    • Make the recurrence tree
    • Find the overlapping subproblems
  2. Applying DP
    • Think how to save the subproblem
    • Use either Top-Down or Bottom-up apporach

Patterns of DP Problems

  1. Fibonacci Pattern

    • When the current state is dependent on mulitple previous states
    • dp[x] = dp[i] + dp[i+1] + ... + dp[x-1]
    • Similar Problems
      1. Count ways to reach the nth stair using step 1, 2 or 3. 
          dp[i] = dp[i-1] + dp[i-2] + dp[i-3]
      2. Find maximum possible stolen value from houses. 
          dp[i] = Max(dp[i-2] + gold[i], dp[i-1))
      3. Count of different ways to express N as the sum of 1, 3 and 4. 
          dp[i] = dp[i-1] + dp[i-2] + dp[i-3]

  2. 0/1 Knapsack

    • Let say we have n items now for any item we can take two decision
      • Take the item (1 State)
      • Leave that item (0 State)
    • Given weights and values of n items, put these items in a knapsack of capacity W to get the maximum total value in the knapsack
      dp[i-1, values]           => Without picking item i
  dp[i, values - weight[i]] => Picking item i only once
  dp[i, values] = Max(dp[i-1, values], cost[i] + dp[i-1, values - weight[i]]
    • One popular vairant is when we have unbounded Knapsack that is we can add items multiple items
      dp[i-1, values]           => Without picking item i
  dp[i, values - weight[i]] => Picking item i (We can pick is multiple times)
  dp[i, values] = Max(dp[i-1, values], cost[i] + dp[i, values - weight[i]]
    • Similar Problems
      1. Min Cost Climbing Stairs. 
          dp[i] = Math.min(dp[i-1], dp[i-2]) + cost[i]
      2. Minimum Path Sum. 
          dp[i][j] = Math.min(dp[i-1][j], dp[i][j-1]) + cost[i][j]
