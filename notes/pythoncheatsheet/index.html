Python Cheat Sheet

List in Python

Declare list

  data = [1,2,3,4]

Add value in list

  # Adding value in x
  data.append(x)

Get at index in list

  data[index]

Get sublist from a to b+1

  dataList = data[a:b+1] // Sub list from [a,b]
  dataList = data[:b+1] // Sub list from [0, b]
  dataList = data[b:]   // Sub list from [b, len(data)-1]

Length of list

  len(data)

Remove value from list

  data.remove(x)  // Remove value x

Remove from any index

  data.pop(index) // Remove value at index

1-D Array with size n

  [0 for i in range(n)]

2-D Array with size m,n

  [[0 for i in range(n)] for j in range(m)]

Iterate over list

  for (index, value) in enumerate(data):
    print(index, end = " ")
    print(value, end = " ")

List Comprehensions

  { value for value in iterable }

  data = ['a', 'b', 'c', 'd']

  data_list = [i for i in data if i != 'a']
  print(data_list)

  data = [i * j for i in [20, 40, 60] for j in [2, 4, 6]]
  print(data)

Sort list

  sort(key, reverse) vs sorted(iterable, key, reverse)
  sort() makes changes to the original sequence

  sorted() returns the new list

  data.sort()
  sorted_data = sorted(data)

  Key(optional): A function that would serve as a key or a basis of sort comparison.
  Reverse(optional): If set True, then the iterable would be sorted
                    in reverse (descending) order, by default it is set as False.

  def keyFunc(data):  
      return data[1]

  keyFuncLambda = lambda data: data[1]
  
  data = [(1, 'utkarsh'), (3, 'singhal'), (2, 'amit')]

  data.sort(key = keyFunc)   
  print(data)  
      
  data.sort(key = keyFunc, reverse = True)  
  print(data)
      
  data1 = sorted(data, key = keyFunc, reverse = True)  
  print(data1)

Dictionary in Python

Declare dict

  map = {'a': 1, 'b': 2, 'c':3}

Get from dict

  map = {'a': 1, 'b': 2, 'c':3}
  value = map['a']

Add in dict

  map = {'a': 1, 'b': 2, 'c':3}
  map['d'] = 4

Dictionary Comprehension

  { key: value for (key, value) in iterable }
  dict = {
    x: x**2 for x in [1,2,3,4,5]
  }
  print (dict)

Lambda

Lambda creation

  """ 
      function without name is called Anonymous functions,
      we can create anonymous functions using lambda
  """
  squareLambda = lambda x: x**2

  def square(x):
    return x**2
  
  y = squareLambda(2)
  y = square(2)
  Both does the same thing

  """ 
    lambda with filter
    filter(function , list):
      function  => True or False
      list      => to list which we need to filter

    Function will return the list after applying function to all the elements
  """
  data = [1,2,3,4]
  list_data = list(filter(lambda x: (x%2 != 0), data))


  """ 
    lambda with map
    filter(function , list):
      function  => return value
      list      => to list which we need to filter

    Function will return the list after applying function to all the elements
  """
  data = [1,2,3,4]
  list_data = list(map(lambda x: x**2, data))

    1) FIFO (Queue)
      q = queue.Queue(maxsize=n)

    2) LIFO (Stack)

    3) PriorityQueue
      q = queue.PriorityQueue(maxsize=n)
      Format of data in this queue is a tuple in the form: (priority_number, data)
      By default it is a min heap
  """
